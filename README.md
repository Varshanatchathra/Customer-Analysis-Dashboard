# Customer-Data-Analysis(Interactive Dashboard creation using Power BI)
## Project Objective
A store needs to analyse the sales report of their store which is located in 5 - different cities .They have the data of top loyalty customers of their stores.By analyzing the data the owner can understand which city is top most and which  city need to improve its sales.
## Dataset used:
- <a href="https://github.com/Varshanatchathra/Customer-Analysis-Dashboard/blob/main/customer%20Analytics.pbix" >Dataset</a>
## Questions
-How many total customers are there?
- Which city contributes the highest revenue?
- Who are the top 5 high-value customers?
- Is loyalty linked to higher purchase amount?
- Which age group spends the most?
- Analyze repeat customers vs purchase value.
